## 📌 Résumé

Développement de compétences (skills) personnalisées pour l'agent IA **Hermes** — un framework open-source pour agents autonomes.

### ✨ Skills créés
- **Obsidian Research Agent** — recherche web automatique + création de notes structurées
- **Blog Research** — rédaction d'articles de blog enrichis avec sources
- **Code Review** — revue de code automatisée avec suggestions

### 🏗️ Architecture
- Skills en Python avec API Hermes
- Intégration GitHub pour push automatique
- Cron jobs pour exécution planifiée
- Kanban pour la gestion des tâches

### 🔥 Impact
Automatisation complète de la recherche et de la rédaction — des centaines d'heures économisées.
