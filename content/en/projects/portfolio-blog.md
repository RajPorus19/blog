My **personal portfolio/blog** built with Hugo and the **hugo-profile** theme. A fast, responsive static site with a custom color palette.

### Design
- Custom green palette (#2EAD67, #E8F5EC)
- Client-side search and filter for projects
- Tags and categories for articles
- Dark mode included
- Bilingual (FR/EN)

### Stack
- **Hugo** — static site generator (blazing fast)
- **hugo-profile** — responsive theme
- **GitHub Pages** — deployment

### Custom features
- Projects page with real-time search 🔍
- Sort by date or name
- Tag filtering
- Articles with table of contents and reading time
