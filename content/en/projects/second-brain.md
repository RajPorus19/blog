A **digital second brain** hosted on Obsidian, connected to GitHub and powered by an AI agent (Hermes). The agent can read, create, and enrich notes autonomously — with automatic push to GitHub.

### Features
- **Autonomous research** — the agent explores the web and creates structured notes
- **GitHub sync** — every modification is committed and pushed automatically
- **Organized structure** — Notes/, Concepts/, Templates/, Anki Cards/
- **Smart wikilinks** — notes are interconnected
- **Full-text search** — via Obsidian search tools

### Stack
- **Obsidian** — read/write interface
- **Git + GitHub** — versioning and backup
- **Hermes Agent** — AI that works on the vault
- **Kanban** — task management for research

### Results
**70+ notes** created on diverse topics: neuroscience, history, geology, philosophy, medicine, politics...
