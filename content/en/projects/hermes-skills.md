Custom skills development for **Hermes** — an open-source framework for autonomous AI agents.

### Skills created
- **Obsidian Research Agent** — automated web research + structured note creation
- **Blog Research** — rich article drafting with sources
- **Code Review** — automated code review with suggestions

### Architecture
- Python skills with Hermes API
- GitHub integration for auto-push
- Cron jobs for scheduled execution
- Kanban for task management

### Impact
Full automation of research and writing — hundreds of hours saved.
